## Usage
```bash
$ git clone git@github.com:brysonbw/qr-code-generator.git

$ cd qr-code-generator

$ yarn 
```

## Developing
```bash
$ yarn dev

Open http://localhost:3000 to view it in the browser.
```

## Building
```bash
$ yarn run build

# To preview build 
$ yarn run preview
```
