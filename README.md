## Contributing
```bash
$ git clone git@github.com:brysonbw/qr-code-generator.git

$ cd qr-code-generator

$ yarn 
```

## Developing
```bash
$ yarn dev

Open http://localhost:3000 to view it in the browser.
```

## Linting
```bash
# check for linting errors
$ yarn lint

# fix linting errors
$ yarn fix-lint
```
